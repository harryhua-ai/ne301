# NE301 Frontend

## Environment Requirements

- **Node.js**: >= 18.0.0 (Recommended: 18.19.0)
- **Package Manager**: pnpm >= 8.0.0

## Install Dependencies

```bash
# Ensure using pnpm
pnpm install
```

## Development

```bash
# Start development server
pnpm dev

# Build project
pnpm build

# Preview build results
pnpm preview
```

## Testing

```bash
# Run tests
pnpm test

# Run test UI
pnpm test:ui

# Run test coverage
pnpm test:coverage
```

## Code Quality

```bash
# Code linting
pnpm lint

# Code formatting
pnpm format

# Auto-fix code issues
pnpm lint:fix
```

## Notes

- This project requires pnpm as the package manager
- Please ensure Node.js version >= 18.0.0
